# micro (terminal editor)

The stack's default `$EDITOR` — modern, mouse-friendly, GUI-like ctrl bindings.

| Key | What |
|---|---|
| `Ctrl-S` / `Ctrl-Q` | save / quit |
| `Ctrl-O` | open a file |
| `Ctrl-C` / `Ctrl-X` / `Ctrl-V` | copy / cut / paste |
| `Ctrl-Z` / `Ctrl-Y` | undo / redo |
| `Ctrl-F` / `Ctrl-N` | find / find next |
| `Ctrl-E` | command bar — `> goto 42`, `> replace a b`, `> set ...` |
| `Ctrl-G` | help |
| `Alt-,` / `Alt-.` | previous / next tab |
| `Ctrl-B` | run a shell command |

Plugins: `micro -plugin install <name>`. Config: `~/.config/micro/settings.json`.
