# eza (modern ls)

Aliased: `ls` = `eza --icons=always --git --group-directories-first`.

| Command | What |
|---|---|
| `eza` | list — icons, git status, dirs first |
| `eza -l` | long view |
| `eza -la` | long + hidden |
| `eza -T` / `eza -TL 2` | tree / tree to depth 2 |
| `eza -l --git` | git status column |
| `eza -l -s size` / `-s modified` | sort by size / mtime |
| `eza -l -r` | reverse sort |
| `eza -l --no-permissions --no-user` | trim columns |
